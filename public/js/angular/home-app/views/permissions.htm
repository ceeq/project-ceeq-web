  • Why So Many Permissions?
  • Ceeq requires 23 permissions. You must be wondering why we require so many permissions. Exactly, when we started we just required just 2-3 permissions like for showing the map etc. But as we increased the number of features, the number of permissions too increase subsequently; because of the beauty of Android OS, without your permission no one can harm your data or device. Still we have tried to reduce those permissions which are not very important, but those used are just to increase the total number of features.

  • Explain all permissions?
    1. We will try to explain all the required permissions in simplest words.

    2. We need the permission to read and write your contacts, messages, call logs, and user dictionary for the sole purpose of backups.

    3. We need the permission to read and write external storage, to save and retrieve data backups files and application preferences on external storage.

    4. We need the permission to access the fine location(GPS) for providing accurate location tracking, whenever you or your device are in trouble.

    5. We need the permission to access camera for taking pictures in case security breach, or when wrong passcodes are inserted to lockscreen.

    6. We need the permission to send and receive messages so that we may check for the remote command send by you, also last we need to send Messages to respond to your remote commands.

    7. We need the permission to check boot completion, and read phone state is required, to check for phone reboot, and to detect SIM changes.

    8. We need the permission to access network state and internet, to check when you are connected to internet; and to upload your login information.

    9. We need the permission to process outgoing calls, to check if you are trying to launch the application in stealth mode from dialpad.

    10. We need the permission to check the running tasks only when the lockscreen is running, to keep the lockscreen always on top, so that no one can minimize the lockscreen and access your data or apps.

    11. We need the permission to get accounts, and read g services for Google+ login & maps, mandatory bt Google itself.

    12. Now, listening to Incoming Messages or SIM changes is completely done by application itself, and so we are not spying you ever, and we will never eat up your balance by sending unnecessary messages.
